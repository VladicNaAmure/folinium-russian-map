## About The Project

Using python folinium map I show approach to draw follium map with HTML for elements. I use branca module to draw elemets as legend on the map:

<img src="/images/legend.png" align="middle" width="100"/>

I use folinium module to draw map by OpenStreetMap data. In layers are region layer to show states and regions name, manager and type task layers.

<img src="/images/layes.png" align="middle" width="100"/>
